## Contenido

Los archivos de esta carpeta forman parte del repositorio y se incluyen tal como están para que la documentación pueda referenciarlos directamente.

Las imágenes cubren, entre otras, estas áreas:

- home y búsqueda pública
- registro e inicio de examen
- práctica y resultados
- panel admin
- certificaciones y plantillas
- importación, auditoría y usuarios

## Uso en la documentación

- Referenciar estas imágenes desde Markdown usando rutas relativas dentro de `docs/public-docs/`.
- Mantener nombres de archivo estables cuando sea posible para no romper enlaces.
- Si se añade una nueva captura, incluirla en esta carpeta y documentar su uso en el archivo correspondiente.

## Nota

Algunos nombres de archivo incluyen espacios o caracteres especiales porque reflejan capturas concretas del flujo documentado. Esto es válido, pero conviene evitar cambios innecesarios en esos nombres una vez publicados.
