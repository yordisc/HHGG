# HHGG

HHGG es una plataforma web satirica de certificaciones creada como pieza de portafolio. Esta carpeta publica agrupa solo la documentacion que ayuda a entender el producto sin exponer el codigo interno.

Sitio de demostracion actual: https://hhgg-5c2f.onrender.com/

## Lo que encontraras

- Una vista breve del producto y su alcance.
- Las guias mas utiles para revisar funcionamiento, despliegue y soporte.
- El contrato de importacion/exportacion que usa el sistema.
- Un mapa claro de que documentos se publican y por que.

## Criterio de publicacion

- Se copian documentos ya existentes que explican el producto de forma util para terceros.
- Se mantiene fuera todo lo que solo aporta contexto interno, operativo o historico.
- La raiz del repositorio publico queda como entrada breve; el contenido curado vive en esta carpeta.

## Aviso

HHGG no es una certificacion real ni sustituye evaluaciones medicas, psicologicas o legales. El objetivo de este material es mostrar alcance tecnico, producto y mantenimiento de una app Laravel moderna.
